# LifeEngine
![Kerbal Visual Enhancements Logo](http://i.imgur.com/9Pupdla.png)

### About project
[**Game of Life**](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) is a cellular automata simulator which defaults to the rules for Conway's Game of Life. It can also be set to use several other GoL rulesets.
